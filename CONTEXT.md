Miners Online is one brand and has multiple Minecraft servers. It has been in operation since 2017, and has had multiple iterations of everything.

Miners Online also develops their own infrastructure and backend services, and has a custom web frontend for all of its online presence.

## Servers

**Minigames**

Version: 1.8-26.1.2
Editions: Java Edition and Bedrock Edition

**Survival**

IP: `survival.mc.minersonline.uk`
Version: 1.8-26.1.2
Editions: Java Edition and Bedrock Edition

Does not operate 24/7, and there is no fixed schedule.

**Modded Creative**

IP: `modded-creative.mc.minersonline.uk`
Version: 1.21.1
Editions: Java Edition

Mods are required. Its a Create mod focused modpack. Does not operate 24/7, and there is no fixed schedule.
