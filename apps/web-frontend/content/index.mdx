## Play Now

Miners Online is a Minecraft minigame server network with a variety of fun and engaging game modes.

Join now using the IP address: `play.minersonline.uk` and experience the fun for yourself!
We support Java Edition 1.8 and above, along with Bedrock Edition on supported platforms.

## Minigames

Minigames are available to everyone, with no membership required.

This game mode is currently in development, but we have plans to release it in the future. Stay tuned for updates!

## Members

Members get exclusive access to classic 2017 experience of Miners Online which includes separate servers, such as:
- **Survival**: A classic Minecraft experience with an economy and land claiming.
- **Modded Creative**: A creative mode server featuring the Create mod and other technical mods, for building and automation enthusiasts.
